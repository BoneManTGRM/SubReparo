# SubReparo Fleet Dashboard Scaffold

This folder defines the future fleet dashboard direction.

Current status: scaffold only.

Fleet mode must use safe summaries and digests by default. Raw local files, private customer data, and endpoint evidence should stay local unless an operator explicitly exports a sanitized bundle.

Planned features:

- endpoint inventory summaries;
- policy compliance summaries;
- risk trend summaries;
- approval status summaries;
- digest-only repair memory exports;
- signed report verification status.

No remote collection agent is implemented here.
