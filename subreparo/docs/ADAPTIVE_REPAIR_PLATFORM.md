# SubReparo Adaptive Repair Platform

## Decision

SubReparo should not be boxed into traditional antivirus.

The strongest product path is:

```text
SubReparo Adaptive Repair Platform
```

Cybersecurity remains an important module, but the broader category is self-healing infrastructure.

## Core promise

SubReparo helps systems stay healthy by continuously observing, detecting, explaining, isolating, repairing, verifying, remembering, and improving.

```text
observe -> detect -> explain -> isolate -> repair -> verify -> remember -> improve
```

## Why this is stronger than cybersecurity alone

Traditional cybersecurity is crowded and trust-heavy.

Adaptive repair is broader:

- AI-agent health and guardrails;
- software project resilience;
- endpoint repair workflows;
- incident response evidence;
- DevOps reliability;
- dependency and supply-chain hygiene;
- website and service monitoring;
- chain-backed repair proofs.

## Product layers

- process review;
- network review;
- startup review;
- browser extension review;
- baseline diff;
- quarantine staging;
- restore;
- dashboard.

### SubReparo Repair

Repair workflow and verification.

- recommended fix plans;
- timeline;
- audit chain;
- incident bundle;
- trends;
- policy;
- rule catalog;
- repair scoring.

### SubReparo Platform

Product packaging.

- simple mode;
- expert mode;
- developer mode;
- family mode;
- enterprise mode;
- installer scripts;
- report export;
- future desktop shell.

### SubReparo Chain

Digest-only repair memory.

- safe event summaries;
- repair status;
- RYE scoring;
- TGRM phase;
- proof that a repair workflow occurred;
- no raw private files on-chain.

## Ideal first customer

The first customer should not be a large enterprise.

Best early users:

- solo developers;
- small cybersecurity shops;
- MSPs;
- AI-agent builders;
- freelance incident-response helpers;
- small software teams.

SubReparo is an adaptive repair platform that gives software, AI agents, and systems a local immune layer for detection, containment, verification, and repair memory.
