# SubReparo Rule Changelog

## 0.2.0

Initial local defensive rule catalog.

- SR-PATROL-001: Encoded command detection.
- SR-PATROL-002: Download-and-run pattern detection.
- SR-PATROL-003: Hidden execution detection.
- SR-PATROL-004: Startup script and executable review.
- SR-PATROL-005: Browser extension sensitive permission review.
- SR-PATROL-006: Launcher and shortcut review.
- SR-PATROL-007: Baseline new watched file detection.
- SR-PATROL-008: Baseline changed watched file detection.
- SR-PATROL-009: Runtime process command-line review.
- SR-PATROL-010: Network connection review.

Rules are defensive-only and intended for systems the user owns or is authorized to inspect.
